Student Data Analysis
This mini project focuses on analyzing student academic data to understand performance trends, subject-wise scores, and overall results.
The project includes basic data cleaning, analysis, and visualization.

--Features--

Student data loading
Data cleaning
Marks analysis
Highest & lowest score detection
Average and percentage calculation
Simple visualizations (if added)

--Dataset Description--

The dataset contains the following columns:
Student ID
Name
Class
Subject marks
Total marks
Percentage
Result (Pass/Fail)
